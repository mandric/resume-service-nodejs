## Setup

```sh
git clone [this repo]
cd resume-service-nodejs
npm install
```

## Running Locally

```sh
npm start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Tests

```
npm test
```

## Deploying to Heroku

```
heroku create
git push heroku master
heroku open
```
